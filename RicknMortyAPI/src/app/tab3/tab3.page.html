Acerca de Acerca de Rick and Morty API Aplicación DEMO desarrollada en iONIC
Este es un ejemplo de una aplicación desarrollada en iONIC (Angular) donde se consume la API Rick and Morty para mostrar en un listado los personajes de la serie antes mencionada y tiene como finalidad el servir de guia para que los desarrolladores nuevos aprendan con un ejemplo sencillo.
